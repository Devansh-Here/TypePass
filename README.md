# 🎹 PASSTYPE — Password Strength Checker

A vintage typewriter-themed password strength analyzer. Type your password and watch it stamp onto paper in real time, with live security analysis and recommendations.

---

## ✨ Features

- **Virtual Typewriter** — keys visually press down as you type, with click sounds
- **Live Strength Meter** — 5-dot indicator updates instantly as you type
- **6 Security Checks** — length, uppercase, lowercase, numbers, special characters, 12+ length
- **Crack Time Estimate** — shows how long it would take to brute-force your password
- **Entropy Score** — calculates password randomness in bits
- **Smart Recommendations** — tells you exactly what to improve
- **Show / Hide Toggle** — mask or reveal your password
- **Vintage Aesthetic** — cream paper, ruled lines, brown keys, typewriter stamp animation

---
